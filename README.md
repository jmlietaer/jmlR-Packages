# jmlR-Packages

------------------
jmlR.graph package
------------------

Create a graph visualization.

Input: 
  txt file

Output: 
  xgml file
  GraphML file

Remark: 
GraphML and xgml output files can be read by yWorks' yEd - https://www.yworks.com/products/yed 
GraphML can be read by Gephi - https://gephi.org/
